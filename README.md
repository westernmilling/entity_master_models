# Entity Models

Shared Entity model library.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'entity_models'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install entity_models

## Usage

First install the migrations with:

```ruby
rails generate entity_models:install
```
